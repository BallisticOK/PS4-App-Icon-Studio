# PS4 App Icon Studio

Windows tool for sending preset and custom PS4 app icon payloads to any compatible app title ID.

The app is no longer branded around the old Vue project. Runtime payload assets now live in [`PayloadKit`](d:\PS4 PKGs\Vue-Icons-main\Vue-Icons-main\PayloadKit), and the app copies them into the build output automatically.

This repo currently patches prebuilt `.elf` payload templates. It does not compile payloads from source yet. Notes for owning that workflow are in [`docs/payload-build.md`](d:\PS4 PKGs\Vue-Icons-main\Vue-Icons-main\docs\payload-build.md).
